Objets Complexes

Utilisateurs : Partie 1

Réaliser cette première partie en n'utilisant pas le constructeur.

  • Créer le script d'un objet utilisateur possédant les caractéristiques suivantes :
    • Nom
    • Prénom
    • Date de naissance
    • Mot de passe
    • Diplômes (tableau) ['BAC', 'DUT']
    • Exepériences utilisateurs (tableau)
    • Hobby (tableau)
    • Codeur (booleen)
    • Compte github
  • Créer ensuite les méthodes permettant :
    • d'afficher les diplomes de l'utilisateur
    • d'afficher les expériences de l'utilisateur
    • d'afficher un message d'accueil personnalisé
    • de calculer son âge automatiquement
    • de comparer une saisie utilisateur avec le mot de passe enregistré (le paramètre sera la saisie) qui renvoie true or false
  • Tester les différentes méthodes et affichage de vos propriétés grâce à un console log

Utilisateurs : Partie 2

Utilisez l'objet que vous venez de créer pour répondre aux questions. Utiliser le HTML et les sélecteurs proposés afin de

  • Permettre l'identification (utilisation des méthodes)
  • Si l'identification est bonne, j'affiche les caractéristiques de l'utilisateur
    • Nom
    • Prénom
    • Je calcule son âge
    • J'affiche son compte Github si il en a un
    • J'affiche ses diplômes / expériences / hobby
  • Si l'identification n'est pas bonne j'affiche la div identification fausse

Utilisateurs : Partie 3

J'utilise le cours et les notions de constructeurs afin de créer plusieurs utilisateurs et de pouvoir comparer les différents affichages de données

  • div contenant l'exercice : #exo1
  • div contenant le formulaire : #form
  • champ nom : #nom
  • champ password : #pwd
  • bouton d'envoie : #login
  • Bloc de réponse (invisible au démarrage) #reponse
  • bloc de réponse nom : #nomR
  • bloc de réponse prénom : #prenom
  • bloc de réponse age : #age
  • bloc de réponse git : #git
  • Liste des diplomes (créations de li à prévoir) : #diploma
  • Liste des expériences (créations de li à prévoir) : #experiences
  • Liste des hobby (créations de li à prévoir) #hobby
  • Bloc contenant l'erreur d'identification (bloc logique + setTmeout à prévoir) : #wrong

        Diaporama : Partie 1

        On se retrouve dans le contexte d'un diaporama qu'on veut gérer grâce à Javascript. Dans cette première partie, on va créer les fonctions et l'objet qui nous permettra de gérer un diaporama simple.

        • Créez un objet qui contiendra les caractéristiques suivantes
          • La cible dans le DOM
          • Le tableau contenant les adresses des images
          • la durée de transition d'une image à l'autre
          • L'indice de l'image visible
          • la méthode qui va permettre de changer l'image affiché
        Les éléments sont à créer dans la div #partie1

        Diaporama : Partie 2

        Cette fois-ci, en conservant ce qui a été réalisé dans la première partie, on va écrire le script du constructeur permettant de créer des objets diaporama.

        • Créez le constructeur Diaporama
        • Initialisez 2 diapos différents
        • Utilisez les selecteurs disponible et le DOM afin d'ajouter dynamiquement ces deux diaporamas (pas les mêmes images ni la même durée)
        Modifiez les propriétés des deux éléments créés en vous inspirant du CSS fournit avec le template

        Les éléments sont à créer dans la div #partie2
