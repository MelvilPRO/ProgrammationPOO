Exercice 2 : Formulaire d'accueil

La propriété permettant d'accéder au contenu d'un formulaire est monInput.value.

  • Ecoutez l'événement clic sur l'envoie du bouton #sendForm
  • Selectionnez les valeurs saisies dans le formulaire aux champs #nom - #prenom - #ddn
  • Sélectionnez le contenant des résultats #resultat
  • Affichez ce contenant en enlevant la classe invisible

Exercice 4 : Les couleurs

Partie 1 : Aléatoire

Nombre aléatoire : Math.floor(Math.random() * 256)

  • Sélectionnez les champs du formulaire #affichRed - #affichGreen - #affichBlue
  • Ecoutez l'événement clic du bouton d'identifiant #clicColor (changer)
  • Générez 3 nombres aléatoires compris entre 0 et 255 inclus
  • Placez ces chiffres dans les différents champs du formulaire
  • Changez le style de #containerColor pour prendre en compte les 3 nombres (maVariable.style.backgroundColor = 'rgb(' + nb1 + ',' + nb2 + ',' + nb3 + ')' )

    • Partie 2 : Formulaire

    • Ecoutez l'événement clic du bouton valider d'identifiant #formColor
    • Récupérez les valeurs contenus dans les champs
    • Grâce à ces valeurs modifiez la couleur du fond de #containerColor
