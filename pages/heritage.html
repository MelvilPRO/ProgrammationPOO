Exercice 1: Les Formes

Créez une classe shape qui permettra de définir les propriétés suivantes :

  • Nom
  • Nombre de côtés
  • Couleur de fond
  • Couleur de la bordure
et les méthodes suivantes (en plus des getters / setters) : Les méthodes seront ici vide mais vous pouvez au moins y glisser un console.log
  • Calcul du périmètre
  • Calcul de l'aire
Créez les classes filles héritant directement ou indirectement de cette classe shape en respectant l'image ci-après. Chaque nouvelle classe fille possédera potentiellement de nouvelles méthodes / propriétés ou modifiera celles existantes.

Exercice 2: Les véhicules

En reprenant le même principe que l'exercice précédent. Travaillez sur une arborescence de classes signifiant les véhicules.
La classe véhicule permettra de définir les propriétés suivantes :

  • Nom
  • Images
  • Année de fabrication
  • Pays d'origine
  • Carburant
et les méthodes suivantes (sans getters / setters) :
  • Afficher image
  • Mouvement
  • Arrêt
